# ApplePy - eCommerce Mobile App

ApplePy es una aplicación móvil de comercio electrónico desarrollada con **Flutter**, inspirada en la estética limpia y minimalista de Apple. El proyecto se centra en ofrecer una experiencia de usuario fluida, utilizando las mejores prácticas de desarrollo y gestión de estado.

## Características

- **Catálogo de Productos:** Visualización de productos en una cuadrícula elegante.
- **Buscador en Tiempo Real:** Filtro dinámico de productos por nombre.
- **Detalles del Producto:** Pantalla dedicada con imágenes en alta resolución y descripciones.
- **Gestión de Carrito:** Sistema funcional para añadir, eliminar y calcular totales.
- **Arquitectura Limpia:** Uso de Provider para la gestión de estado de forma eficiente.
- **Diseño Responsive:** Adaptado para diferentes tamaños de pantalla en Android e iOS.

## Tecnologías Utilizadas

- [Flutter](https://flutter.dev/) - Framework de UI.
- [Dart](https://dart.dev/) - Lenguaje de programación.
- [Provider](https://pub.dev/packages/provider) - Gestión de estado.
- [Google Fonts](https://fonts.google.com/) - Tipografía personalizada.

## Instalación y Configuración

Para ejecutar el proyecto localmente:

1. **Clona el repositorio:**
   ```bash
   git clone [https://github.com/TU_USUARIO/applepy_ecommerce.git](https://github.com/TU_USUARIO/applepy_ecommerce.git)
2. **Entra a la carpeta del proyecto:**
cd applepy_ecommerce
3. **Instala las dependencias:**
flutter pub get
4. **Ejecuta la aplicación:**
flutter run
**Requisitos Previos**
- Flutter SDK (Versión estable)
- Android Studio / VS Code
- JDK 17 (Recomendado)
